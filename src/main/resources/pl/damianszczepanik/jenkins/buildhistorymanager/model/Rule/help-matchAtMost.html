Description

Number of matched jobs after which the rule is not processed. Then this condition is met, only other rules are processed.

Use cases

Gives possibility to limit number of changes performed by given rule.
-1 - there is no limitation, all jobs will evaluated
0 - zero jobs will be evaluated
1..n - when conditions match n times, no more jobs will be processed by this rule

Warning!

Zero value can be used to temporary disable to rule without deleting whole configuration
