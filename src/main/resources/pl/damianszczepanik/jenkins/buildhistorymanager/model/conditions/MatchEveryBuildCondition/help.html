Description

Matches every build without any condition.

Use cases

Helpful when need to perform action for every build from the history

Warning!

Use this action wisely because if wrong build will be matched to this action then recovering deleted build might be impossible.

Plugin is logging a lot about what is doing and why. More about how to find logs find here. Find entries tagged as pl.damianszczepanik.jenkins.buildhistorymanager.
